About


Hi,Welcome to Prognosist!

Prognosis is a web application that uses machine learning algorithms to identify patterns in the dataset and predict a disease. Users can input symptoms and the system predicts accurately. We have also tried to improve the dataset by allowing doctors to add data to the dataset. Our system can mainly be used if the doctors are unable to predict a disease. We believe that our application can help in significantly improving the accuracy and speed of diagnosis.

Currently the application is in beta stage. In the upcoming updates we will try to improve analysis to try and predict if there is an outbreak. With a lot of input from doctors, we believe our application can easily detect increase in certain symptoms and prevent an outbreak like another COVID.
